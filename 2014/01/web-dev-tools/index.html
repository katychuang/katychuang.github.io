Essential Tools for Web Development

Posted on January 5, 2014 with labels: PyData

Here are some tools I picked up along the last few months that drastically improved the experience of maintaining multiple projects in production. It’s by no means a comprehensive list and I’d love to keep finding more tools.

TMUX/Vim

The PyData.org website lives on a ubuntu server, configured with NGinX and gUnicorn. In the course of deploying updates, I found myself using the same two tasks in two different directories, with switching back and forth too much. Tmux lets you have multiple sessions or what I like more is multiple panes. On one pane I’ll pull the latest code and double check file permissions. The other pane is dedicated to restarting the server. A few keystrokes ctrl-b o lets me switch from one to the other.

Vim comes preinstalled with ubuntu linux, and so if I had to change the settings on the server (i.e. turn off the Django debug mode), I can change the server copy instead of my laptop’s local settings.

GitX

This little gem is a lifesaver in writing good commit messages. Unlike many of the other git GUI clients available, it is developed by open source and is therefore free.

You can preview and cherry pick proper ones and then have a neater commit log. It’s also great for viewing commit logs in different branches without having to switch over your code. I like using a GUI interface a lot more than the comandline git.

Chrome Extensions

For front end development, I really like using whatfont and web developer to check font faces and sizes, and to check various DOM element renderings. Chrome web developer tools is awesome as well for debugging javascript errors.

For API development, I like using Postman to check json formatted data. It’s a great way to test various header request settings.

As a researcher, I use onetab to make it easy to keep track of many many open tabs and import/export pages and switch among computers. I also use delicious to keep track of bookmarks.

Theming

There are a ton web design resources available, from startbootstrap examples to articles listing free resources to magazines.
