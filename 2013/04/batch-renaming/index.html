Batch renaming files

Posted on April 2, 2013 with labels: tutorial, pydata

After PyCon, I had several photos to upload for the PyLadies group. They had the right word in the filename but I wanted to rename the files properly to notate the year and proper capitalization for archival purposes. Python makes it easy to do simple file name manipulation.

With one command “python rename.py” the script changes multiple files.

Here’s the resulting file names.

Below is the python script for changing image names. You can download it as a file from this github gist.

Title: Batch Image File Renaming
Author: Kat Chuang @katychuang
Copied from http://stackoverflow.com/questions/9835243/python-rename-multiple-image-files


import os

# Create a list of files from the current directory who's last 4 characters
# as lowercase are either '.jpg'

files = [f for f in os.listdir('.') if f[-4:].lower() == '.jpg']

DRYRUN = True

for (index, filename) in enumerate(files):
    extension = os.path.splitext(filename)[1]

    # Define new file name pattern
    newname = "PyCon2013-%0d%s" % (index, extension)

    if os.path.exists(newname):
        print "Cannot rename %s to %s, already exists" % (filename, newname)
        continue

    if DRYRUN:
        print "Would rename %s to %s" % (filename, newname)
    else:
        print "Renaming %s to %s" % (filename, newname)
        os.rename(filename, newname)
